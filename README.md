# PDFViewer
Android PDF Viewer based on MuPDF optimized for working together with my [PDF Creator](https://github.com/scoute-dich/PDFCreator) app.

### Permissions:
"android.permission.WRITE_EXTERNAL_STORAGE" to read PDf files
